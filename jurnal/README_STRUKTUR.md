## Cara Akses PDF

Semua file dari **PMC (PubMed Central)** bisa diakses gratis:
1. Klik link PMC di kolom "Sumber"
2. Klik tombol **"PDF"** di halaman artikel
3. Download langsung

Untuk file **"Original"**, cek di folder asli atau akses via:
- Google Scholar: [scholar.google.com](https://scholar.google.com)
- ResearchGate: [researchgate.net](https://www.researchgate.net)
- Sci-Hub: Gunakan DOI (gunakan dengan bijak)
